#  NariRaksha

### AI-Driven Women Safety and Alert System

##  Overview

NariRaksha is an AI-powered women safety web application designed to provide real-time protection and emergency assistance. The system enables users to send instant SOS alerts, share live location, and detect danger situations using Artificial Intelligence.

---

##  Features

*  One-click SOS alert system
*  Real-time location tracking
*  AI-based risk detection (text analysis)
*  Full-stack MERN application
*  Secure data handling with MongoDB

---

##  Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **AI Service:** FastAPI (Python)
